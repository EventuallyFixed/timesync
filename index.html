About TimeSync

Disappointed because there was no incremental backup system bundled with my Western Digital EX4100 NAS, I decided to embark on a project to build one.

It is heavily influenced by the BackInTime project, although it's a completely independent development that has been aimed at this range of devices.

The target was to create a web-based, mobile-friendly application, making use of only the applications that are bundled with the Personal Cloud device:

  • php
  • rsync
  • SQLite

The application also relies on these technologies to function:

  • HTML
  • SQLite
  • JQuery
  • JSON
  • Bootstrap (and StartBootstrap)
  • CSS

Thanks to Ravishanker Kusuma for the backend code from his jQuery Upload File Plugin.

A big thanks to the many sites and - of course - to the countless number of contributors over the years, that were used as a resource for tips and tricks, and without which the timeline of this project would have been greatly extended.
